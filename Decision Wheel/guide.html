Menu Rush 사용 가이드

추천 설정부터 결과 공유와 관련 이미지 확인까지, 자주 사용하는 흐름을 실제 화면 순서에 맞춰 정리했습니다.

1. 기본 추천 흐름

  1. 식사 문화 기준 선택: 먹고 싶은 국가 또는 현재 생활권에 가까운 국가를 선택합니다.
  2. 식사 시간 선택: 자동을 사용하거나 아침, 점심, 간식, 저녁, 야식, 전체 중 하나를 직접 고릅니다.
  3. 추천 시작: 슬롯이 멈추면 중앙에 결정 메뉴가 표시됩니다.
  4. 후보 변경: 메뉴 섞기를 누르면 같은 조건에서 후보 순서를 새로 섞습니다.

Tip: 자동 모드는 방문 국가가 아니라 현재 선택한 식사 문화 국가의 현지 시간을 기준으로 판단합니다.

2. 결과 공유 방법

결과 공유

지원되는 모바일 브라우저에서는 기기의 공유 메뉴가 열립니다. 메신저, SNS, 메모 앱 등을 선택할 수 있습니다.

결과 복사

추천 국가, 식사 시간과 결정 메뉴가 포함된 주소를 복사합니다. 상대방이 링크를 열면 같은 결과를 확인할 수 있습니다.

Tip: TikTok이나 YouTube 설명란에는 결과 링크와 함께 “오늘 뭐 먹지?”처럼 짧은 질문을 적으면 내용을 이해하기 쉽습니다.

3. 관련 이미지 검색

추천 결과 아래에는 Unsplash와 Pexels의 관련 이미지가 표시됩니다. 슬롯 중앙의 결정 메뉴를 누르면 Unsplash, Pexels, Google Images에서 원본 국가 언어의 메뉴명으로 직접 검색할 수도 있습니다.

서비스명을 누르면 새 탭으로 열리고, 옆의 창 아이콘을 누르면 별도 창으로 열립니다. 외부 검색 결과의 정확성과 이용 조건은 각 제공자가 관리합니다.

4. 추천을 더 잘 활용하는 팁

선택지가 너무 많을 때

전체 대신 현재 식사 시간 하나만 선택하면 후보가 줄어 결정하기 쉬워집니다.

새로운 음식을 찾을 때

익숙하지 않은 국가를 선택하고 전체 메뉴를 사용해 보세요. 결정 메뉴를 눌러 이미지로 먼저 확인할 수 있습니다.

여럿이 함께 고를 때

각자 한 번씩 추천을 돌린 뒤 공유 링크를 모아 최종 후보를 정하면 의견을 빠르게 좁힐 수 있습니다.

식단 제한이 있을 때

추천은 알레르기나 종교적 식단을 자동 판별하지 않습니다. 메뉴 재료와 조리법을 반드시 별도로 확인하세요.

5. 자주 묻는 질문

추천 결과를 어떻게 공유하나요?

메뉴 결정 후 결과 공유를 누르거나 결과 복사로 주소를 전달하면 됩니다.

자동 식사 시간은 어떻게 정해지나요?

선택한 국가의 현지 시각과 해당 국가에 설정된 식사 구간을 사용합니다.

언어를 바꾸면 검색어도 번역되나요?

화면 문구와 메뉴 표시는 번역되지만 이미지 검색은 식사 문화 국가의 원본 메뉴명을 사용합니다.

왜 관련 이미지가 정확하지 않을 수 있나요?

외부 이미지 제공자의 검색 데이터가 부족하거나 음식명이 다른 뜻으로 해석될 수 있습니다. 이때 결정 메뉴의 직접 검색 링크를 이용하세요.

추천 결과는 매번 같은가요?

아니요. 같은 조건에서도 후보를 무작위로 섞기 때문에 결과가 달라질 수 있습니다.

서비스 이용에 비용이 발생하나요?

현재 모든 메뉴 추천 기능은 무료이며 회원가입이 필요하지 않습니다.
