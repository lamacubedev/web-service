오늘의 메뉴 결정을 가볍게

Menu Roulette Rush는 설치와 가입 없이 사용할 수 있는 무료 음식 추천 도구입니다. 선택한 국가의 식사 문화와 현지 시간을 반영해 지금 먹기 좋은 후보를 보여줍니다.

만든 이유

“오늘 뭐 먹지?”라는 질문은 단순해 보여도 매일 반복되면 결정 피로가 됩니다. Menu Roulette Rush는 선택지를 무작정 늘리기보다 국가와 식사 시간을 먼저 좁힌 뒤, 슬롯 방식으로 한 가지 메뉴를 제안합니다.

주요 기능

국가별 메뉴40개 국가의 실제 음식명을 국가 원문으로 관리합니다.
현지 시간 자동 선택선택 국가의 시간대를 기준으로 현재 식사 구간을 판단합니다.
직접 필터아침, 점심, 간식, 저녁, 야식 또는 전체 메뉴를 직접 선택할 수 있습니다.
관련 정보 탐색추천 메뉴의 이미지와 Unsplash, Pexels, Google 이미지 검색 링크를 제공합니다.

추천 방식

추천은 선택한 국가와 식사 시간에 해당하는 중복 없는 메뉴 후보를 무작위로 섞어 결정합니다. 자동 모드에서는 선택 국가의 현지 시간을 사용하고, 전체 모드에서는 해당 국가의 모든 등록 메뉴를 사용합니다.

서비스 특징

빠른 사용첫 화면에서 설정부터 추천, 결과 공유까지 끝낼 수 있습니다.
접근성데스크톱과 모바일에서 같은 흐름으로 사용할 수 있도록 설계했습니다.
유지보수메뉴 데이터, 번역, 화면 기능을 분리해 지속적으로 개선합니다.
투명성무작위 추천의 한계와 외부 이미지 검색 사용을 명확히 안내합니다.

운영 방향

실제 음식명과 식사 문화의 정확성을 우선하며, 사용자 피드백을 반영해 메뉴 분류와 번역 품질을 개선합니다. 메뉴 수를 인위적으로 늘리기보다 확인 가능한 이름을 유지하는 것이 원칙입니다.
